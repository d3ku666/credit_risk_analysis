# credit_risk_analysis
Credit risk analysis of 50,000 loans using Python, Pandas and Machine Learning to predict loan default probability.

# Credit Risk Analysis Project

This project analyzes a dataset of 50,000 loans to identify
factors associated with loan default.

## Objectives

- Explore credit risk variables
- Identify drivers of loan default
- Build a predictive model

## Tools Used

- Python
- Pandas
- Seaborn
- Scikit-learn

## Model Performance

- Accuracy: ~86%
- AUC: ~0.69
- KS Statistic: ~0.29

## Key Findings

- Higher leverage increases probability of default
- Lower credit scores correlate with higher risk
- Certain sectors present higher default rates
